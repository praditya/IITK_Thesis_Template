\section{Null Space}

\textbf{Null Space}:
The null space of a matrix $A$ is defined as the set of all the vectors $\mathbf{v}$ such that $A\mathbf{v} = 0$, i.e.,
\begin{equation}
N(A) = \{\mathbf{v}|A\mathbf{v}=\mathbf{0}\}
\end{equation}
